## Инструкция по сборке

Требуется компиялтор с поддержкой C++17

Для сборки проекта необходимо выполнить следующую команду
```
cd /tmp
git clone ...
cd copy
mkdir build && cd build
cmake ..
cmake --build .
```

### Запуск

```
copy 1.ex 23.exe
```
